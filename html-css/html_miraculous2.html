FRONT-END

Conceito

Front-End é a parte visual e interativa de um site ou aplicação.

É tudo aquilo que o usuário vê e com o que interage na tela.

Envolve:

- Layout

- Cores

- Botões

- Animações

- Responsividade

- Experiência do usuário (UX)

Trabalha diretamente no navegador.


HTML

O que é?

HTML significa HyperText Markup Language.

É a linguagem responsável por estruturar o conteúdo da página.

Curiosidade:

A primeira página criada na história da internet ainda pode ser acessada atualmente.

Ela define:

- Títulos

- Parágrafos

- Imagens

- Links

- Botões

- Listas

É a “estrutura” do site (como o esqueleto do corpo).

CSS

O que é?

CSS significa Cascading Style Sheets.

É responsável pela aparência visual do site.

Curiosidade:

O CSS permite criar animações e até efeitos 3D sem usar JavaScript.

Controla:

- Cores

- Fontes

- Espaçamentos

- Tamanhos

- Animações

- Layout

É como se fosse a “roupa” e o design do site.
