FRONT-END

Conceito

Front-End é a parte visual e interativa de um site ou aplicação.

É tudo aquilo que o usuário vê e com o que interage na tela.

Envolve:

- Layout

- Cores

- Botões

- Animações

- Responsividade

- Experiência do usuário (UX)

Trabalha diretamente no navegador.


HTML

O que é?

HTML significa HyperText Markup Language.

É a linguagem responsável por estruturar o conteúdo da página.

Ela define:

- Títulos

- Parágrafos

- Imagens

- Links

- Botões

- Listas

É a “estrutura” do site (como o esqueleto do corpo).

Curiosidade:

A primeira página criada na história da internet ainda pode ser acessada atualmente.


CSS

O que é?

CSS significa Cascading Style Sheets.

É responsável pela aparência visual do site.

Controla:

- Cores

- Fontes

- Espaçamentos

- Tamanhos

- Animações

- Layout

É como se fosse a “roupa” e o design do site.

Curiosidade:

O CSS permite criar animações e até efeitos 3D sem usar JavaScript.


JavaScript

O que é?

JavaScript é a linguagem que dá interatividade e lógica ao site.

Permite:

- Validação de formulários

- Botões interativos

- Modais

- Atualização dinâmica de conteúdo

- Jogos no navegador

É o “cérebro” da página.

Curiosidade:

O JavaScript foi criado em apenas 10 dias por Brendan Eich em 1995.


Frameworks Front-End

Frameworks são ferramentas que facilitam e aceleram o desenvolvimento.

Eles oferecem:

- Estrutura pronta

- Componentes reutilizáveis

- Organização do código

- Melhor desempenho


React

React é uma biblioteca JavaScript para criar interfaces modernas e dinâmicas.

Características:

- Baseado em componentes

- Virtual DOM

- Muito usado no mercado

Criado por:

Meta (antigo Facebook)


Angular

Angular é um framework completo para desenvolvimento de aplicações web.

Características:

- Usa TypeScript

- Estrutura robusta

- Muito usado em sistemas grandes

Criado por:

Google


Vue

Vue.js é um framework progressivo e mais simples de aprender.

Características:

- Leve

- Fácil integração

- Sintaxe simples

Criado por:

Evan You
